# FinDev Mobile

Projeto desenvolvido na Semana Omnistack 10 da Rocketseat.

Versão mobile da aplicação FinDev que tem por finalidade encontrar desenvolvedores por geolocalização.


# Tecnologias:
* React Native
* Axios


# Conceitos:
* React Navigation
* Components
* Props
* State
* API REST
* MapView component
* WebView component
